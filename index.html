HTML

Linguagem de marcação usada para estruturar páginas web com títulos, parágrafos, imagens, links e outros elementos.

CSS

Define o estilo visual das páginas web, como cores, fontes, espaçamentos e posicionamento de elementos.

JavaScript

Linguagem de programação que adiciona interatividade às páginas web, como cliques, animações e validações.

TypeScript

Superset do JavaScript que adiciona tipagem estática, ajudando a evitar erros e melhorar o código.

Node.js

Ambiente que permite executar JavaScript no servidor, usado para back-end, APIs e automações.

React

Biblioteca JavaScript para construir interfaces de usuário com componentes reutilizáveis e estado dinâmico.

GitHub

Plataforma para hospedar, versionar e colaborar em projetos de código usando o sistema de controle Git.

Vite

Ferramenta de build rápida para projetos front-end, que acelera o desenvolvimento com hot reload e configuração simples.

Python

Linguagem de programação versátil, usada em web, automações, dados e inteligência artificial, conhecida por sua simplicidade.
